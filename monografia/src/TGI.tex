\begin{resumo}
 Aplicações web podem apresentar vulnerabilidades de segurança. Scanners de vulnerabilidades são ferramentas capazes de identificar essas vulnerabilidades. O objetivo deste trabalho é avaliar a segurança de aplicações web em relação à vulnerabilidades do tipo \emph{XPath Injection} e também avaliar a eficácia de \estrangeiro{scanners} de vulnerabilidades na detecção dessas vulnerabilidades.
\end{resumo}

\begin{abstract}
 Web application can have security vulnerabilities. Web Application vulnerability scanners are tools capable of identifying those vulnerabilities. This work tries to evaluate the security of web application with respect of XPath Injection vulnerability and also assess the effectiveness of vulnerability scanners to detect these vulnerabilities. 
\end{abstract}
