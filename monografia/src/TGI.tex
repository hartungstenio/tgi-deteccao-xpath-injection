\begin{resumo}
 Aplicações web podem apresentar brechas de segurança. Scanners de vulnerabilidades são ferramentas capazes de identificar essas brechas. O objetivo deste trabalho é avaliar a segurança de aplicações web em relação à vulnerabilidade do tipo \emph{XPath Injection} e também avaliar a eficácia de scanners de vulnerabilidades na detecção dessas vulnerabilidades.
\end{resumo}

\begin{abstract}
 Web application can have security holes. Web Application vulnerability scanners are tools capable of identifying those holes. This work tries to evaluate the security of web application with respect of XPath Injection vulnerability and also assess the effectiveness of vulnerability scanners to detect these vulnerabilities. 
\end{abstract}
